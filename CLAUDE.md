# VCI Workshop

- **Stack:** Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS v4, ESLint (`eslint-config-next`).
- **Entry points:** Routes and UI live under `src/app/`; static files in `public/`.
- **Commands:** `npm run dev` (dev server with Turbopack), `npm run build`, `npm start`, `npm run lint`.

When implementing features, prefer existing patterns in `src/app` and match TypeScript + Tailwind usage already in the template.
